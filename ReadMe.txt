# 📝 Blog Post Generator

A tool that automatically generates blog posts using Groq API (Llama 3). Just enter a topic, and AI writes a complete blog post for you.

## Features
- Generate blog posts from any topic
- Choose tone (Friendly / Professional / Humorous)
- Adjust length (Short / Medium / Long)
- Save as Markdown file

## Usage
```bash
python blog_generator.py
